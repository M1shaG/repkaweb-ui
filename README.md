# R.E.P.K.A Web UI

This project was developed for the assignment from the university: "Create a SRS(Software Requirements Specification) document and implement a module from it. " The module presents a simple web user interface.

## Stack 

Vite + React + JavaScript + AntDesign

    Vite - build tool
    React - frontend library
    JavaScript - programmming language
    Antdesign - React UI library
